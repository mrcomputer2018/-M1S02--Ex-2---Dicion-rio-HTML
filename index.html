Dicionario de Tags HTML

Construindo um dicionário HTML com os principais elementos que nos conhecemos.

Elementos HTML

Os principais elementos HTML que conheço são:

<HTML>

- Todos os documentos HTML devem começar com uma declaração de tipo de documento: <DOCTYPE html>.

- O elemento HTML é o elemento raiz de uma página HTML e contém todo o conteúdo da página, como cabeçalhos, parágrafos, imagens, hiperlinks, tabelas, listas, etc.

- O próprio documento HTML começa com <html> e termina com </html>.

- A parte visível do documento HTML está entre <body> e </body>.

<head>

- Define informações sobre o documento.

- O elemento <head> contém metadados do documento, como título, links para estilos, scripts e outras metainformações.

<title>

- Define o título do documento

<body>

- Define o corpo do documento

- O elemento <body> contém todo o conteúdo de um documento HTML, como texto, hiperlinks, imagens, tabelas, listas, etc.

<h1> a <h6>

- Define títulos de diferentes niveis de importãncia.

- O elemento <h1> define o título mais importante. O elemento <h6> define o título menos importante.

<p>

- O elemento <p> define um parágrafo.

- Um parágrafo sempre começa em uma nova linha e é um bloco de texto.

<ul>

- Define uma lista não ordenada

- O elemento <ul> define uma lista não ordenada. Um item de lista é definido com o elemento <li>.

<li>

- O elemento <li> é usado para definir um item em uma lista.

- É um elemento de nível de bloco.

<strong>

- O elemento <strong> é usado para definir um texto importante.

- O texto dentro do elemento <strong> é exibido em negrito.

<a>

- O elemento <a> define um hiperlink, que é usado para vincular de uma página para outra.

- O atributo href especifica o URL da página para a qual o link vai.

<img>

- O elemento <img> é usado para incorporar uma imagem em um documento.

- O elemento <img> é um elemento vazio, o que significa que ele não tem um elemento de fechamento.

- Ele contém apenas atributos e não contém texto ou elementos filho.

- O atributo src especifica o URL da imagem.
